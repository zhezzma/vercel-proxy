Vercel Proxy

一个简单可靠的请求代理服务，支持 URL 编码、错误处理、超时保护等特性。

基本用法

GET https://[your-vercel-domain]/api?url=https://api.example.com/data
支持所有 HTTP 方法（GET、POST、PUT、DELETE 等）

特性

  • 完整的 CORS 支持
  • 自动 URL 解码
  • 30 秒超时保护
  • 错误处理和友好提示
  • 支持编码后的 URL

示例

编码的 URL：

GET https://[your-vercel-domain]/api?url=https%3A%2F%2Fapi.example.com%2Fdata%3Fkey%3Dvalue

POST 请求：

POST https://[your-vercel-domain]/api?url=https://api.example.com/submit
Content-Type: application/json

{
    "data": "example"
}

错误处理

所有错误都会返回标准的 JSON 响应：

{
    "error": "错误描述信息",
    "status": 400-599
}

状态码说明

  • 400 - 请求参数错误（如缺少 url 参数）
  • 405 - 方法不被允许
  • 502 - 目标服务器连接错误
  • 504 - 请求超时
